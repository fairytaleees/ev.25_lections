git -  распределение система контроля версий     Это система для отслеживания и ведения истории изменений файлов, в вашем проект.
репозиторий - хранилище вашего кода и истории его изменений 
GitHub - веб сайт для размещения git - репозиториев в совместной разработке проеков.

Команды Git:

 1. git init - она создает новый локальный гит репозитор у вас на пк , в той директории в которой была прописана команда. (команда вводится один раз в начале)
 2. git add- команда для добавления файлов в систему отслеживания гита ,тпосле этого изменения начинают отслеживаться гитем.
 3. git commit -m '<comment>' - как только достигаем определенного момента в разработке ,то всегда мы сохраняем комментируем изменения в локальный репозиторий    
 
 4. git remote add- это команда для того чтобы связать наш локальный репозиторий с удаленным репозиторием , то есть доавляет подключенте к репозиторий на гитхабе.
 
 git push -  команда для отправки всех ваших изменений на локальном репозитории в удаленным репозиторием , то есть добавляет подключение к репозиторий на гитхабе.
           git remote add<name : origin > < ссылка_на_репо:ssh>
           
    5.   git push - команда для отправки всех ваших изменений на лшокальном репозитории в удаленный репозиторий 
        git push origin main
